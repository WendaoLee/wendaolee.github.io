在实际 Web 开发过程中，我逐渐明白一个网站的字体对于整个网站的呈现效果会是不一样的。前一段时间我刚好看到了 Oleh 的博文 [Quick guide to web typography for developers](https://sinja.io/blog/web-typography-quick-guide)，里面正好讲到了网站的字体。这勾起了我的兴趣。于是就有了这样的一篇博文。

让我感兴趣的点有：

1. 有哪些字体是主流浏览器，或者说操作系统都带有的？
2. 字体可以怎样分类？
3. 不同的字体格式的区别是？
4. 在Web开发里，我们该如何导入外部字体？

这篇博文会慢慢讲述我的探索过程。
